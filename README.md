# Star Carpet Demo

Демонстрационная копия редизайна [Star Carpet](../star-carpet): тот же дизайн и основные сценарии, но **без реального наполнения**.

## Что внутри

- Полный UI: главная, каталог, карточка товара, корзина, избранное, сравнение, примерка
- Страницы: доставка, опт, о компании, контакты, блог
- **12 шаблонных товаров**, 6 коллекций, 3 статьи
- Контакты и реквизиты — плейсхолдеры (`+7 (000)…`, «адрес заказчика»)
- Картинки — Unsplash / локальный placeholder

## Чего нет (намеренно)

- Импорт YML / полный каталог (~3800 SKU)
- Реальные телефоны, адрес, ИП
- SEO-редиректы со старого сайта
- Длинные scraped-тексты

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm start
```

Для GitHub Pages:

```bash
GITHUB_PAGES=true npm run build
```

`basePath` — `/star-carpet-demo`.
